About

Contra Tunes seeks to help musicians plan their next jam session and share their plans with fellow musicians.
Upload your ABC-notation music and instantly share it with others.

Credits

Website by Nicholas Gower
Initial tune collection by Phil Katz

Built with

Frontend

Interactivity: htmx
Sheet Music Display: abcjs
ABC Editor: abctools-django (Fork of abctools by Michael Eskin )
Other Libraries: JQuery

Backend

Framework: Django
Production Server: Gunicorn, Nginx
Database: PostgreSQL
User Authentication: Django-all-auth

Server

Web Host: AWS EC2
Email Host: Zoho
OS: Ubuntu
Container: Docker
